## Props
<ArgsTable of={Box} />

The `Box` component is a layout component (also see [every-layout.dev](https://every-layout.dev/)).
It provides a base layout that is often extended by unique styles depending on the situation they are applied to.

## Using Style Attributes
Style Props is a way to extend component styles via props.
Example given, you can extend the components `padding` by adding a prop `px` which adds `padding-inline` (or `padding-left` and `padding-right`) to the styled component.
You can provide any valid css value to this prop, though, it is recommend to stick to the design tokens given by the theme and stored in the css `:root`.
